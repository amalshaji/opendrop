02 / gap
/problem lost context
The gap

Static artifacts are moving faster than review workflows.

Agents, designers, and front-end teams can produce HTML drops quickly. The handoff still happens through screenshots, Slack files, temporary buckets, and links that lose context.

Feedback is detached from the page, pixel, viewport, and version.

Private previews are either too heavyweight or too improvised.

Teams need a review room, not another rebuild pipeline.

OD
03 / product
/solution artifact-first
The product

OpenDrop turns any static output into a governed preview.

upload

Folder or zip

Browser upload or `npx opendrop upload ./dist`.

validate

Safe artifact

Paths, sizes, duplicates, counts, and root index checked before publish.

publish

Immutable version

Latest URL stays stable. Version URLs preserve the exact drop.

review

Inline comments

Coordinates, highlights, replies, and resolved state stay attached.
